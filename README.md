## Installation & Usage
<hr/>
<br/>

Installation:
```bash
npm i -g @xfxstudios/create-base-app
```
<br/>
<p>Using</p>

```bash
npx create-base-app
```
<p>and follow wizard.</p>

## Flags:

install all node dependencies from template
```bash
npx create-base-app --install true
```
